Program

Learning to code in Python is easier and more useful than learning to write in English. In fact you can write programs in the Python language that requires less syntax than other programming languages. A lot humans like to beat around the bush when it comes to programming by using complex programming environments such as IDLE, ATOM, Eclipse, etc. However, the easiest way to use Python is using a text editor (GNU Nano for Linux, or NotePad for Windows), just make sure to save your file with the extension .py which is standard for Python files. For example, we can write a program called example.py, and then we run it in the command line by entering the command python example.py

The simplest Python program that you can write is with a simple print statement. print(thing) where the command print will tell the computer to print to the screen the value of the variable 'thing'.

    Data

    Some basic ideas in data for Python are: variables, lists, tuples, and dictionaries. A variable is a string of characters that holds a value. For example, dollar = 100 * penny the variable dollar holds the value of 100 pennies. A variable can be any string of characters like dollar, however, you can't start a variable with a special character such as * or /, nor can you start a variable with an integer such as 2 or 8.

    Perhaps the most widely used data structure is the list which is a set of items of the same type, such as [1,3,89,29] or [soda, coffee, water]. In this case, the lists holds integers and strings. tuples are a special type of list because a tuple is a list, but the items in this list is another list which holds 2 items only. An example of a tuple is [[soda,1.25],[water,0.99],[coffee,3.25]] which is a list of items that are lists with 2 items each.

    A dictionary is a data structure that is similar to a list because it holds items, and it is similar to a tuple because each item is a set of 2 things. These 'things' are called a key and a value. In python you use dictionaries just as you would use a word dictionary, in order to find the a definition (value) you need to know what word (key) to find. Python keeps a dictionary as an unordered set of 'key:value' pairs. you define a dictionary as simple as my_dictionary = {} this statement creates an variable my_dictionary that holds an empty dictionary.
